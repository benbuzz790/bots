Inline Comment Tokenization:
`python
# Input: "x = 1  # comment"
# Tokenized: "x = 1; TOKEN_hash_0"
# Token map: {"TOKEN_hash_0": "  # comment"}  # Preserves spacing
`
Reuniting Logic:
`python
def _should_reunite_comment(code_line, comment_line, token_map):
    for original in token_map.values():
        if original.strip() == comment_line.strip():
            # Check if token has leading spaces before #
            stripped = original.lstrip()
            if stripped.startswith('#') and len(original) > len(stripped):
                return True  # Was inline, should reunite
    return False
`
